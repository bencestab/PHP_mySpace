REST API webarchitektúra

API (Application Programming Interface): Az API (alkalmazásprogramozási felület) egy szoftverkomponens vagy szolgáltatás által nyújtott interfész, amely lehetővé teszi más szoftverek számára, hogy kommunikáljanak vele és kihasználják a funkcionalitását. Az API-k lehetnek webes vagy más típusú interfészek, de itt most a webes API-kre fókuszálunk.

A webes API-k lehetővé teszik a különböző szoftverek vagy alkalmazások közötti kommunikációt a weben keresztül. Az API-k különböző végpontokat (endpoints) definiálnak, amelyeken keresztül kéréseket lehet küldeni, és válaszokat lehet kapni. Általában a kérések és válaszok JSON (JavaScript Object Notation) formátumban történnek, de más formátumokat is használhatnak, például XML-t.

Az API-k segítségével az alkalmazások képesek adatokat lekérdezni, módosítani vagy más műveleteket végezni más szolgáltatásokkal vagy adatforrásokkal. Például egy szociális média platform lehetővé teheti, hogy egy harmadik féltől származó alkalmazás lekérdezze a felhasználók profiladatait vagy posztjait az API-ján keresztül.
